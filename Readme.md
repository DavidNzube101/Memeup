# MemeUp 🎨

MemeUp is an interactive Reddit game where users compete by submitting the funniest captions for memes. The bot posts a new meme daily, and users vote on the best captions to climb the leaderboard!

---

## How It Works 🚀

1. **Daily Meme Post**: A bot posts a new meme with the challenge: "Caption this!"
2. **Caption Submissions**: Users submit their funniest captions as comments.
3. **Voting**: Users upvote their favorite captions.
4. **Leaderboard**: The top-voted captions are displayed on a leaderboard.
5. **New Round**: The cycle resets daily with a new meme.
